# SLFlipForge Execution + Deployment Bundle

Deployment-ready founder MVP starter for SLFlipForge.

Included:
- Next.js starter structure
- Mobile bottom navigation
- Flip Scanner
- Market Intelligence
- AI Listing Engine
- Photo to Listing
- Save to Inventory
- Inventory Manager
- Buyer Messages
- Analytics
- Supabase schema starter
- Environment example

Quick start:
1. Unzip
2. Run npm install
3. Copy .env.local.example to .env.local
4. Add your Supabase values
5. Run supabase/schema.sql
6. Run npm run dev
7. Deploy to Vercel
